# `awsinfo ecs task [-s] [cluster-filter]* -- [task-filter]*`

Describe a task.

## Options

* `-s`: Match against stopped tasks

## First filter matches against

* Cluster Name

## Second filter matches against

* Task Arn
* TaskDefinitionArn
* ContainerInstanceArn
* LastStatus
* Group
* Cpu
* Memory
* LaunchType
